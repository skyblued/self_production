2018 阿里云 API 应用创新大赛

2018年 阿里云 API 应用创新大赛是由阿里云主办的第二届 API 开发者大赛。大赛基于广义的 API 经济理念，邀请大家一起体验 API 在应用间串联上的更大作用，而不仅仅是 API 即服务、即能力、即数据。参赛者可以选择大赛在 Serverless、API 实战、物联网、视频、人工智能、企业服务 6 个方面制定的赛题，也可以完全自主命题，为解决某个场景的某个问题进行创新开发。优秀作品及战队除了赢得丰厚奖励，还将能够直面投资人，展示你的想法。还有机会认证为阿里云官方合作伙伴，与阿里云共建。诸多奖励和权益，等待有想法、有行动力的你来领取！
